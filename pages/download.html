Как установить язык на свой компьютер?

Пропишите в терминал команды для скачивания и установки:

git clone https://github.com/Dis-Studio/Iron-Lang
cd Iron-Lang
make libs (ЕСЛИ ОТСУТСТВУЮТ ПАКЕТЫ: nasm, gcc, binutils, nodejs)
make install (ЖДИТЕ ПОКА КОМПИЛЯТОР СКОМПИЛИРУЕТСЯ И УСТАНОВИТСЯ)

Пока-что доступно только на Linux.
